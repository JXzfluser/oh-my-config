📖 使用指南

🌐 LLM Wiki (全局知识库)

点击左侧「LLM Wiki」进入全局知识库，所有项目共享：

  • INDEX.md - 全局目录
  • LOG.md - AI 探索发现记录
  • ENTITIES.md - 实体索引
  • CONCEPTS.md - 概念索引

🏠 项目管理

点击项目切换，悬停显示删除按钮。输入新项目名创建。

📁 文件类型

  • Wiki - 项目知识库 (README、NOTES)
  • 规范 - CLAUDE.md、AGENTS.md 等
  • 配置 - 服务配置 (JSON)

⌨️ 快捷键

  • Cmd/Ctrl + S - 保存

📤 同步

「导出」复制到剪贴板，「下发」同步到目录。

🔗 CLI 命令

omc projects      # 列出项目
omc init myapp   # 初始化项目
omc get myapp CONFIG  # 获取配置
omc sync myapp  # 同步到目录
